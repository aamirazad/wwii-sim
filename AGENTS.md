# AGENTS.md

## Setup commands
- Install deps: `bun install`
- Do not start the dev server
- Run linter `bun lint`
- Write linter suggestions `bun lint --write`

## Project Setup

- In the `apps` folder, the backend and the frontend are separated
- The backend is an elysia typescript backend in the `api` folder
- The frontend is a nextjs project in the `web` folder
